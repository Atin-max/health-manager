# Healthcare Hub

Healthcare Hub is a web-based healthcare and fitness application developed using HTML, CSS, and JavaScript. The project aims to promote a healthy lifestyle by providing users with workout plans, diet recommendations, and health-related calculators through a simple and user-friendly interface.

## Features

- 🏋️ Workout Planner
  - Fat Loss and Muscle Gain plans
  - Beginner, Intermediate, and Advanced levels
  - Dynamic weekly workout generation

- 🥗 Diet Guide
  - Weight Loss, Muscle Gain, and Balanced Diet plans
  - Healthy food alternatives
  - Random meal suggestions
  - Water intake tracker

- 🧮 Health Calculator
  - BMI Calculator
  - BMR Calculator
  - Daily Calorie Requirement Calculator
  - Water Intake Calculator

## Technologies Used

- HTML5
- CSS3

## Project Structure

- Dashboard Module
- Workout Module
- Diet Module
- Calculator Module

## Purpose

The project was developed as part of the Bachelor of Computer Applications (BCA) curriculum to demonstrate the use of front-end web technologies in creating an interactive healthcare and fitness platform.
